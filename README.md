# Raja Automobile

Welcome to the Raja Automobile web platform. This is a decentralized application built on the Internet Computer Protocol (ICP) with a React frontend interface.

## 🚀 Features
* Modern React 19 user interface
* Smooth 3D graphics elements via Three.js
* Fully responsive design for mobile and desktop
